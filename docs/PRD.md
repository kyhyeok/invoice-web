# QuoteSync MVP PRD

## 핵심 정보

**목적**: 프리랜서·에이전시가 노션에서 작성한 견적서를 클라이언트에게 깔끔한 웹 뷰어와 PDF로 손쉽게 공유할 수 있게 한다
**사용자**: 노션으로 견적서를 작성하는 프리랜서·디자이너·에이전시 운영자, 그리고 링크로 견적서를 받아 확인하는 클라이언트

---

## 사용자 여정

```
1. [랜딩 페이지]
   ↓ "시작하기" 버튼 클릭

2. [로그인 페이지]
   ↓ 이메일/비밀번호 입력 또는 회원가입 링크 클릭

   [신규 사용자] → [회원가입 페이지] → 가입 완료 후 자동 리디렉션
   [기존 사용자] → 로그인 성공 시 자동 리디렉션
   ↓

3. [대시보드 페이지] (운영자 전용)
   ↓ "노션 연동" 버튼 클릭 (미연동 상태일 때)

   [미연동] → [노션 연동 페이지] → 연동 완료 후 대시보드로 복귀
   [연동 완료] → 견적서 목록 바로 표시
   ↓

4. [대시보드 페이지] (견적서 목록)
   ↓ 특정 견적서의 "공유 링크 생성" 버튼 클릭

5. [공유 링크 생성 모달] (대시보드 내)
   ↓ 링크 생성 완료

6. [대시보드 페이지] → 운영자가 링크를 복사하여 클라이언트에게 전달
   ↓ (클라이언트 흐름 분기)

7. [견적서 뷰어 페이지] (클라이언트 접근 - 로그인 불필요)
   ↓ "PDF 다운로드" 버튼 클릭

8. [완료] → PDF 파일 저장
```

## 기능 명세

### 1. MVP 핵심 기능

| ID       | 기능명                | 설명                                                                                | MVP 필수 이유                                      | 관련 페이지        |
| -------- | --------------------- | ----------------------------------------------------------------------------------- | -------------------------------------------------- | ------------------ |
| **F001** | 노션 Integration 연동 | Integration Token으로 노션 워크스페이스 연결                                        | 견적서 데이터를 가져오기 위한 핵심 연결 고리       | 노션 연동 페이지   |
| **F002** | 견적서 목록 조회      | 연동된 노션 데이터베이스에서 견적서 페이지 목록을 가져와 표시                       | 운영자가 공유할 견적서를 선택하는 핵심 기능        | 대시보드 페이지    |
| **F003** | 공유 링크 생성        | 견적서별 고유 토큰 기반 공유 URL 생성 (만료 일시 옵션 포함)                         | 클라이언트에게 견적서를 전달하는 유일한 수단       | 대시보드 페이지    |
| **F004** | 견적서 웹 뷰어        | 공유 링크 접속 시 노션 데이터를 기반으로 견적서를 브랜드 친화적 레이아웃으로 렌더링 | 클라이언트가 견적서를 확인하는 핵심 화면           | 견적서 뷰어 페이지 |
| **F005** | PDF 다운로드          | 웹 뷰어 화면을 PDF 파일로 변환하여 저장                                             | 클라이언트가 견적서를 보관·제출하기 위한 필수 기능 | 견적서 뷰어 페이지 |

### 2. MVP 필수 지원 기능

| ID       | 기능명              | 설명                                         | MVP 필수 이유                               | 관련 페이지                    |
| -------- | ------------------- | -------------------------------------------- | ------------------------------------------- | ------------------------------ |
| **F010** | 기본 인증           | 이메일·비밀번호 회원가입 / 로그인 / 로그아웃 | 운영자 식별 및 데이터 보호를 위한 최소 인증 | 로그인 페이지, 회원가입 페이지 |
| **F011** | 노션 연동 상태 관리 | Integration Token 저장·수정·해제             | 운영자별 노션 연결 정보 관리                | 노션 연동 페이지               |
| **F012** | 공유 링크 관리      | 생성된 링크 목록 확인 및 링크 비활성화(삭제) | 운영자가 공유 범위를 통제할 수 있어야 함    | 대시보드 페이지                |

### 3. MVP 이후 기능 (제외)

- 소셜 로그인 (Google, GitHub OAuth)
- 노션 OAuth 연동 (Integration Token만 지원)
- 견적서 조회 통계 (클라이언트 접근 횟수 등)
- 커스텀 브랜딩 (로고 업로드, 색상 변경)
- 견적서 승인/거절 기능
- 이메일 알림
- 다국어 지원

## 메뉴 구조

```
QuoteSync 내비게이션
├── 홈 (랜딩 페이지)
└── 인증 (비로그인 시)
    ├── 로그인 - F010
    └── 회원가입 - F010

운영자 메뉴 (로그인 후)
├── 대시보드
│   └── 기능: F002 (견적서 목록 조회), F003 (공유 링크 생성), F012 (링크 관리)
├── 노션 연동
│   └── 기능: F001 (Integration Token 연결), F011 (연동 상태 관리)
└── 로그아웃

공개 접근 (로그인 불필요)
└── 견적서 뷰어 (/quote/[token])
    └── 기능: F004 (웹 뷰어), F005 (PDF 다운로드)
```

## 기술 스택 (최신 버전)
